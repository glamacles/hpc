# GLIDE ice-flow engine (from source). Pure-Python setuptools package: the CUDA
# kernels ship as package-data and are JIT-compiled at runtime by CuPy's bundled
# NVRTC, so no nvcc/toolkit is needed at install. Pulls cupy-cuda12x (matches the
# cu124 baseline) + pyproj transitively.
# Tracks main: each `terraform apply` rebuilds the VMs and installs main as of
# that moment, so apply once the course env is set rather than mid-course.
glide @ git+https://github.com/glide-ism/glide.git@main
